# 家族グループ表示（取得）

## スコープ

### 含める

- ログインユーザーの所属家族グループ取得
- 所属家族のメンバー一覧表示
- `/family` 画面（未所属時は作成フォーム、所属時は家族名 + メンバー一覧）

### 含めない（別 PR / 別ブランチ）

- 家族への参加（join）
- 脱退（leave）
- グループ名変更（update）
- メンバーの表示名・メール（profiles 未実装のため userId のみ）
- 招待 UI

## 完了条件

- [ ] 未所属ユーザーは `/family` で作成フォームが表示される
- [ ] 所属ユーザーは家族名とメンバー一覧が表示される
- [ ] `/family/new` は `/family` へリダイレクトされる
- [ ] 関連 unit test が通る
