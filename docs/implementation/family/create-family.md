# 家族グループ作成（登録）

## スコープ

### 含める

- 家族グループの新規作成（`families` INSERT）
- 作成者本人を `family_members` に追加（最初のメンバーは本人のみ）
- 未所属ユーザーのみ作成可（1 ユーザー 1 家族）
- Server Action + 作成画面（`/family/new`）

### 含めない（別ブランチ / 別 PR）

- 既存家族への参加（join）
- 脱退（leave）
- グループ名変更（update）
- メンバー一覧表示（get members）
- 招待リンク・招待 UI

## 完了条件

- [ ] ログインユーザーが家族名を入力してグループを作成できる
- [ ] 作成後、作成者が `family_members` に 1 件登録されている
- [ ] 既に所属しているユーザーは作成できない
- [ ] 関連 unit test が通る
