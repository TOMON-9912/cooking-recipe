# プロフィール作成（登録）

## スコープ

### 含める

- `profiles` テーブル新規作成（表示名・既定アイコンキー）
- ログインユーザーのプロフィール初回作成
- プロフィール未作成ユーザーは `/top` から `/profile/new` へ誘導
- 作成画面（表示名入力 + 既定アイコン集から選択）
- Server Action + 関連 unit test

### 含めない（別 PR / 別ブランチ）

- プロフィール更新（update）
- プロフィール取得画面（get / 表示専用ページ）
- 家族メンバー一覧への表示名反映
- 画像アップロードによるアイコン設定

## 完了条件

- [ ] 未作成ユーザーが表示名とアイコンを選んでプロフィールを作成できる
- [ ] 作成済みユーザーは `/profile/new` にアクセスすると `/top` へリダイレクトされる
- [ ] 関連 unit test が通る
